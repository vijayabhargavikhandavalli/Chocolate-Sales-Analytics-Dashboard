# 📊 Chocolate Sales Analytics Dashboard

## Overview

This project is an interactive Power BI dashboard developed to analyze business performance through data visualization and KPI reporting. The dashboard provides valuable insights into revenue trends, profitability, regional performance, and key business metrics, enabling data-driven decision-making.

## Objectives

* Monitor business performance using key KPIs.
* Analyze trends and patterns in business data.
* Identify opportunities for growth and improvement.
* Support strategic decision-making through interactive reporting.

## Tools Used

* Power BI
* Power Query
* DAX
* Microsoft Excel

## Dashboard Features

* KPI Cards
* Trend Analysis
* Revenue & Profit Analysis
* Interactive Filters & Slicers
* Dynamic Visualizations
* Executive-Level Reporting

## Skills Demonstrated

* Data Cleaning & Transformation
* Data Modeling
* DAX Calculations
* Dashboard Development
* Data Visualization
* Business Intelligence
* Data Storytelling

## Key Outcomes

* Improved understanding of business performance metrics.
* Generated actionable insights through interactive dashboards.
* Enabled efficient monitoring of KPIs and trends.
* Demonstrated data-driven decision-making capabilities.
